### Requirements

### Execution

1. Clone this repository:

   ```sh
   git clone https://github.com/ventura-v/rocketseat-nlw-expert-python-flask.git
   cd rocketseat-nlw-expert-python-flask
   ```

2. Create a virtual environment and activate it.

3. Install dependencies:

   ```sh
   pip install -r requirements.txt
   ```

4. Run the Flask app (check the main entry point, for example):

   ```sh
   flask run
   # or
   python app.py
   ```

5. Open the URL printed in the terminal (usually `http://127.0.0.1:5000`).
